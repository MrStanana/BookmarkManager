# BookmarkManager
Cross-platform bookmark database manager.
Built using Qt.

## Features
- Categories
- Search
- Database encryption
- Minimize to system tray

## Dependencies
Qt library (Windows):
- Qt5Core.dll
- Qt5Gui.dll
- Qt5Widgets.dll
- platforms\\qwindows.dll
